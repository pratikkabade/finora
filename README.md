# Finora

**Finora** is a modern money tracking application designed to help users manage their personal finances with ease. 

Whether you're looking to budget your expenses, track your income, or analyze your spending habits, Finora has got you covered.

## About

- This entire project is **developed in under 5 hours using the power of Vibe Coding**. 

   - Vibe Coding is an **AI-powered coding assistant that helps developers build applications quickly and efficiently**. 
   - By leveraging advanced AI technologies, Vibe Coding enables rapid development cycles, allowing developers to focus on creating innovative solutions without getting bogged down by repetitive coding tasks.

- At later stage, due to lack of quality fixes from Vibe Coding, the project was manually fixed and enhanced to achieve the desired functionality.
